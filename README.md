# 💼 Personal Portfolio Website

My Personal Portfolio Website is a modern, fully responsive portfolio built using pure HTML and CSS.  
This project highlights my academic background, technical skills, leadership experience, and project work in a clean and structured layout.

🔗 Live Demo: [View](https://dsdipu.github.io/portfolio/)  
🔗 Repo Link: [View](https://github.com/dsdipu/portfolio)

---

## 📌 Overview

This is a static personal portfolio website designed to professionally present my profile, achievements, and technical expertise.  

The website focuses on structured layout design, responsive implementation, and clean UI using only HTML and pure CSS — without any external frameworks or JavaScript.

It serves as a portfolio-ready project to showcase my development skills and academic journey.
